# SymbolicMath

**SymbolicMath** — это Ruby gem для символьных математических преобразований с фокусом на работу с полиномами.

Gem позволяет выполнять операции символьного дифференцирования, интегрирования, раскрытия скобок и решения уравнений для полиномиальных выражений.

## Возможности

- **Парсинг полиномов** из строки во внутреннее представление
- **Символьное дифференцирование** полиномов по заданной переменной
- **Символьное интегрирование** полиномов (степенное правило)
- **Раскрытие скобок** для произведений полиномов
- **Решение уравнений** (линейные и квадратные)
- **Упрощение выражений** (приведение подобных членов)

## Поддерживаемые выражения
Gem работает с полиномами, содержащими:
<ul>
<li>Переменные: `x`, `y`, `z` (одна буква)</li>
<li>Числа: целые и дробные (например, `2`, `3.5`)</li>
<li>Операции: `+`, `-`, `*`, `/`, `^` (степень)</li>
<li>Скобки: для группировки</li>
</ul>

### Примеры поддерживаемых выражений
<ul>
<li>2*x^3 + 5*x^2 - 3*x + 1</li>
<li>3*x^2 * 2*x</li>
<li>(x + 2)*(x - 3)</li>
<li>(2*x^2 + 3*x)^2</li>
</ul>
