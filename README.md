# SymbolicMath
SymbolicMath — это Ruby gem для символьных математических преобразований.
Он позволяет выполнять операции символьного дифференцирования, интегрирования, раскрытия скобок и решения уравнений

## Возможности
<ul>
<li>Парсинг выражений из строки во внутреннее представление</li>
<li>Символьное дифференцирование по заданной переменной</li> 
<li>Базовое интегрирование</li>
<li>Упрощение выражений (свёртка констант, приведение подобных)</li>
<li>Раскрытие скобок</li>
<li>Решение уравнений (линейные и квадратные)</li>
<li>Подстановка значений в выражения</li>
</ul>
